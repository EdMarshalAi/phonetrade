# Яндекс.Метрика — настройка целей и конверсий (счётчик 91129028)

Счётчик подключён в админке (**Настройки → Интеграции → Яндекс.Метрика**, включён) и грузится на витрине при согласии посетителя на cookie-аналитику. Веб-визор, карта кликов и e-commerce включены в коде. Цели отправляются через `reachGoal` автоматически — в Метрике их нужно один раз **создать** с такими же идентификаторами.

## 1. Создать цели (Метрика → Настройки счётчика → вкладка «Цели» → «Добавить цель»)

Тип цели для всех — **«JavaScript-событие»**, поле «Идентификатор цели» = значение из таблицы:

| Идентификатор | Что считает | Где срабатывает |
|---|---|---|
| `order` | **Оформленный заказ (главная конверсия)** | успешная отправка заказа из корзины; передаёт `order_price` + валюту RUB |
| `checkout_start` | Начало оформления | вход в шаг оформления в корзине |
| `add_to_cart` | Добавление в корзину | кнопка «В корзину» (передаёт цену товара) |
| `trade_in` | Заявка на trade-in | успешная отправка квиза trade-in |
| `search` | Поиск по сайту | отправка поискового запроса (param `search_query`) |
| `call` | Клик по номеру телефона | любой клик по ссылке `tel:` |
| `messenger` | Переход в мессенджер/соцсеть | клик по WhatsApp / Telegram / VK |

> Идентификаторы вводить ровно как в таблице (латиницей, без пробелов). Название цели — любое понятное («Заказ», «Звонок» и т.д.).

## 2. Включить выручку по заказам (e-commerce)

1. Метрика → **Настройки счётчика → вкладка «Электронная коммерция»** → включить, контейнер данных — **`dataLayer`** (в коде уже шлётся `ecommerce.purchase` с `revenue` и товарами).
2. После этого в отчёте **«Электронная коммерция → Заказы»** будет видна сумма выручки, товары и количество.
3. Дополнительно у цели `order` Метрика подхватит `order_price` — в отчётах по целям будет «Стоимость».

## 3. Веб-визор и карта кликов

Уже включены в init (`webvisor:true, clickmap:true`). В настройках счётчика проверьте, что **Веб-визор** активен (Метрика → Настройки → Веб-визор), при необходимости включите запись форм.

## 4. (Опционально) Составная цель / воронка

Чтобы видеть путь до покупки, создайте **составную цель** из шагов:
`search`/`add_to_cart` → `checkout_start` → `order`. Это даст воронку конверсии в заказ.

## Проверка

- Открыть сайт, **принять cookie** (иначе Метрика не грузится — это требование 152-ФЗ), добавить товар, оформить заказ.
- Метрика → **Отчёты → Конверсии** — цели начнут накапливаться; в режиме реального времени видно сразу.
- Если целей нет: проверьте, что счётчик включён в админке и дано согласие на аналитику.

## Кастомные коды (пиксели/виджеты)

Любой сторонний счётчик/пиксель/чат добавляется без кода: **Настройки → Интеграции → «Добавить интеграцию»** → вставить код → выбрать `<head>` или перед `</body>`. Появится карточка с тумблером вкл/выкл.
