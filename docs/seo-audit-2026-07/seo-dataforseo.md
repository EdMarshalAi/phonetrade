# SEO-аудит: seo-dataforseo — PhoneTrade

> Дата: 2026-07-02. Скил `seo-dataforseo` v2.2.0 — живые SEO-данные через DataForSEO MCP.
> Платный (за каждый вызов). Требует установки extension. У нас **НЕ подключён**.

## Что проверяет/делает скил

Живые данные поиска через 10 API-модулей (79+ MCP-инструментов): SERP (Google/Bing/Yahoo/YouTube/Images), keyword research (объём, difficulty, intent, тренды), беклинк-профили, on-page (Lighthouse + парсинг), конкуренты, бизнес-листинги, **AI-видимость** (ChatGPT scraper, LLM mention tracking), domain analytics. Ценообразование — от $0.001/вызов (SERP regular) до $0.05 (keyword ideas, domain competitors, listings, AI/GEO). Пресеты бюджета: Conservative $2/день … Aggressive $50/день.

## Как применить к нам — и стоит ли подключать

**Честный вывод: для PhoneTrade DataForSEO сейчас НЕ нужен. Приоритет — Яндекс, а DataForSEO — это в первую очередь Google-данные (location_code по умолчанию 2840=US, движок Google).** Ключевое ограничение: **DataForSEO не даёт яндексовскую частотность и позиции по-настоящему** — а именно они решают на нашем рынке. То, ради чего его берут (объём ключей, difficulty, беклинки конкурентов), мы уже покрыли бесплатно:

| Что даёт DataForSEO | Наша бесплатная альтернатива (уже есть) |
|---|---|
| Keyword volume | **Яндекс.Вордстат** (реальная частотность по Белгороду) — `seo-keywords-all.csv` (9000+), `-belgorod.csv` |
| Позиции в SERP | **Topvisor** (регион=Белгород, Яндекс+Google) — `topvisor-keywords.txt` (125 шт) |
| Конкуренты (ключи, структура) | Ручная разведка — `seo-competitors-recon-2026-06.md` (6 конкурентов) |
| Индексация, показы, CTR | **Яндекс.Вебмастер** + **Google Search Console** (оба verified) |
| On-page / Lighthouse | Локально + PageSpeed Insights (бесплатно, см. `seo-google.md`) |
| Беклинки | Яндекс.Вебмастер («Внешние ссылки») + Bing Webmaster — бесплатно |

**Единственное, что DataForSEO делает лучше и чего у нас нет — AI-видимость (GEO):**
- `ai-scrape <query>` ($0.05) — что реально отдаёт ChatGPT web search по нашим запросам, какие домены цитирует.
- `ai-mentions <keyword>` ($0.05) — как LLM упоминают бренды/домены по «купить айфон Белгород», кто в топе цитирований.

Это единственный сценарий, где точечное подключение оправдано: **разово проверить, цитируют ли нас (и кого цитируют) ChatGPT/Perplexity по нашим гео-Apple запросам**. Но и это можно частично закрыть руками (спросить сами ChatGPT/Алису «где купить iPhone в Белгороде»).

## Рекомендации

### 🔴 Критично
- Ничего критичного — отсутствие DataForSEO не блокирует ни один наш SEO-процесс.

### 🟡 Важно
- **Если когда-нибудь подключать — только для AI-видимости (GEO), пресет Conservative ($2/день).** Разовый прогон `ai-scrape` + `ai-mentions` по 10–15 гео-Apple запросам = ~$1.5. Даст картину, где нас нет в ответах ИИ. Это дополняет наш GEO-фундамент (llms.txt, answer-first, FAQPage) реальным замером.
- Для Google-SERP конкурентов дешевле `serp_live_regular` ($0.001), но данные US-центричны — для Белгорода бесполезны без смены location_code, а российских локаций у DataForSEO меньше и точность ниже Вордстата.

### 🟢 Nice-to-have
- `serp-images` для анализа image-SERP — но у нас фото уже в Storage с alt-текстами, ниша не даёт большого трафика.
- `backlinks competitors` — интересно, но Яндекс.Вебмастер по ссылкам бесплатен, а покупные ссылки мы всё равно не берём (риск фильтра для локального магазина).

## Быстрые победы
1. **Не подключать.** Сэкономленный бюджет/время → в отзывы и контент (реальный рычаг для локала).
2. Ручная проверка AI-видимости бесплатно: спросить ChatGPT, Алису, Perplexity «где купить iPhone 17 в Белгороде» — записать, упоминают ли нас. Это 90% пользы `ai-scrape` за $0.
3. Если нужна разовая картина AI-цитирований — Conservative-пресет, ≤$2, только `ai-scrape`/`ai-mentions`, потом отключить.

## Google vs Яндекс
- **DataForSEO сильно смещён в Google/западный поиск.** Для яндексо-приоритетного проекта это фундаментальный минус: главный движок (Яндекс) он покрывает слабо, а Вордстат + Вебмастер + Topvisor дают то же бесплатно и точнее по РФ.
- **AI-видимость** (единственная уникальная ценность) частично разделена по движкам: ChatGPT/Perplexity (западные, тянут из Bing) vs Алиса/Нейро (Яндекс, тянет из Яндекс.Бизнеса/Карт). DataForSEO меряет в основном первую группу. Для второй (важнее нам) — работает off-site в Яндекс.Бизнесе, а не DataForSEO.
- **Итог:** для Google — если уж мерить AI-цитаты, DataForSEO полезен разово; для Яндекса — бесплатный стек Вордстат/Вебмастер/Topvisor покрывает всё нужное.
